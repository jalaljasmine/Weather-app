#  Weather App

A simple weather application built using HTML, CSS, and JavaScript.

## Features
- Search weather by city name
- Displays temperature in Celsius
- Shows current weather condition
- Responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
- Weather API

# How to Run
Open index.html in any web browser.
